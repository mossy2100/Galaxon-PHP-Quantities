# Terminology

| Term               | Definition                                                                                                                                                                                                |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Unit               | A single-symbol measurement unit, e.g. *m*, *s*, *N*, *lbf*.                                                                                                                                              |
| Base unit          | A unit that cannot be expanded into more fundamental units, e.g. *m*, *s*, *kg*, *lb*.                                                                                                                    |
| Expandable unit    | A unit that substitutes for a combination of base units, e.g. *N* can be expanded to *kg*·*m*/*s*². This includes all SI named units plus a few others like *lbf*, *cal*, *eV*, *kn*, etc.                |
| SI base unit       | One of the 7 fundamental SI units: *m*, *kg*, *s*, *A*, *K*, *mol*, *cd*, plus *rad*, which, despite technically being an SI derived unit, is treated as an SI base unit for the purpose of this package. |
| English base unit  | A base unit within the imperial and US customary systems of units that is not also an SI unit: *lb*, *ft*, *°R* (degrees Rankine), or ° (degrees of angle).                                               |
| Common base unit   | Additional base units defined (for the purpose of this package) for data and currency quantities: *B* (bytes) and *XAU* (troy ounces of gold).                                                            |
| SI named unit      | An SI unit named after a person, e.g. newton (*N*), joule (*J*), watt (*W*).                                                                                                                              |
| Unit term          | A unit with optional prefix and/or exponent, e.g. *km*, *KiB*, *m²*, *km³*.                                                                                                                               |
| Derived unit       | Zero or more unit terms combined via multiplication/division, e.g. *m*/*s*, *kg*·*m*/*s*².                                                                                                                |
| Metric prefix      | A decimal scaling prefix, e.g. *k* (10³), *M* (10⁶), *m* (10⁻³).                                                                                                                                          |
| Engineering prefix | A metric prefix that represents a power of 1000.                                                                                                                                                          |
| Binary prefix      | A binary scaling prefix for data units, e.g. *Ki* (2¹⁰), *Mi* (2²⁰).                                                                                                                                      |
| Unit system        | A system of units, such as SI, Imperial, US customary, etc.                                                                                                                                               |
