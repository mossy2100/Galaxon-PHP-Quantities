# Money

The `Money` class extends `Quantity` with currency support, including exchange rate services and locale-aware formatting.

## Setup

Currency units and conversions are loaded dynamically from cached data files. Before using money quantities, refresh the exchange rates:

```php
use Galaxon\Quantities\Currencies\CurrencyService;
use Galaxon\Quantities\QuantityType\Money;

// Refresh exchange rates (fetches from an online service if needed).
CurrencyService::refresh();

$price = new Money(100, 'USD');
echo $price->to('EUR');  // converted at current rates
echo $price->to('GBP');
```

## Exchange rate services

The package includes adapters for several exchange rate APIs. Set the preferred service before refreshing:

```php
use Galaxon\Quantities\Currencies\CurrencyService;
use Galaxon\Quantities\Currencies\Frankfurter;

// Frankfurter (ECB data, free, no API key required) — the default.
CurrencyService::setService(new Frankfurter());

// Other available services (require API keys):
// - ExchangeRateApi
// - OpenExchangeRates
// - CurrencyLayer
// - Fixer
```

For services that require an API key:

```php
use Galaxon\Quantities\Currencies\ExchangeRateApi;

$service = new ExchangeRateApi('your-api-key');
CurrencyService::setService($service);
CurrencyService::refresh();
```

## Currency formatting

The `Money` class overrides `__toString()` to use PHP's `NumberFormatter` for locale-aware currency display:

```php
CurrencyService::setLocale('en_US');

$price = new Money(1234.56, 'USD');
echo $price;  // $1,234.56

CurrencyService::setLocale('de_DE');
$price = new Money(1234.56, 'EUR');
echo $price;  // 1.234,56 €
```

If no locale is set, the standard `format()` method is used as a fallback.

## Currency codes

Currencies use [ISO 4217](https://en.wikipedia.org/wiki/ISO_4217) three-letter codes as their unit symbols (e.g. `USD`, `EUR`, `GBP`, `JPY`, `AUD`). All currencies belong to the `Financial` unit system.

The base currency for internal conversions is the troy ounce of gold (`XAU`), chosen because it is the most commonly traded international currency.
