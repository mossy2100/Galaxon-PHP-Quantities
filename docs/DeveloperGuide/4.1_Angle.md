# Angle

The `Angle` class extends `Quantity` with trigonometric functions, angle normalization, and DMS (degrees, minutes, seconds) support.

## Trigonometric Functions

All six trigonometric functions are available as methods. They convert the angle to radians internally before computing:

```php
use Galaxon\Quantities\QuantityType\Angle;

$angle = new Angle(45, 'deg');

$angle->sin();   // 0.7071067811865...
$angle->cos();   // 0.7071067811865...
$angle->tan();   // 1.0

$angle->sec();   // 1.4142135623731...
$angle->csc();   // 1.4142135623731...
$angle->cot();   // 1.0
```

The methods work with any angle unit — degrees, radians, gradians, or turns:

```php
$rightAngle = new Angle(M_PI / 2, 'rad');
$rightAngle->sin();  // 1.0
$rightAngle->cos();  // 0.0 (effectively zero)
```

### Handling undefined values

When a trigonometric function is undefined (e.g. `tan(90°)`, `sec(90°)`, `csc(0°)`), the method returns `INF` or `-INF` rather than producing a division error. The sign is determined by the direction of approach:

```php
$right = new Angle(90, 'deg');
$right->tan();   // INF
$right->sec();   // INF

$zero = new Angle(0, 'deg');
$zero->csc();    // INF
$zero->cot();    // INF
```

## Angle Normalization with `wrap()`

The `wrap()` method normalizes an angle to a standard range:

```php
$angle = new Angle(270, 'deg');

// Signed range: (-180, 180] (default)
echo $angle->wrap();          // -90 deg

// Unsigned range: [0, 360)
echo $angle->wrap(false);     // 270 deg

$angle2 = new Angle(720, 'deg');
echo $angle2->wrap();         // 0 deg
echo $angle2->wrap(false);    // 0 deg
```

The range boundaries depend on the unit:
- Degrees: (-180, 180] signed, \[0, 360) unsigned
- Radians: (-π, π] signed, \[0, τ) unsigned

## Approximate Equality

`Angle` overrides `approxEqual()` to compare in radians with a default absolute tolerance of `1e-9` radians:

```php
$a = new Angle(180, 'deg');
$b = new Angle(M_PI, 'rad');
$a->approxEqual($b);  // true
```

## DMS Parts

Angles support part decomposition into degrees, arcminutes, and arcseconds by default:

```php
$angle = new Angle(45.504200, 'deg');

$parts = $angle->toParts();
// ['sign' => 1, 'deg' => 45, 'arcmin' => 30, 'arcsec' => 15.12]

echo $angle->formatParts();
// 45° 30′ 15.12″

$restored = Angle::fromParts(['deg' => 45, 'arcmin' => 30, 'arcsec' => 15]);
echo $restored;  // 45.504167 deg
```
